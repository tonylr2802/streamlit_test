# Análisis de Vehículos Usados

Esta aplicación web fue desarrollada con Streamlit para explorar un conjunto de datos de anuncios de venta de vehículos usados.

## Funcionalidades

- Visualización de la distribución del kilometraje (odómetro) mediante un histograma.
- Visualización de la relación entre el precio y el kilometraje mediante un gráfico de dispersión.
- Gráficos interactivos creados con Plotly.

## Tecnologías utilizadas

- Python
- Pandas
- Streamlit
- Plotly

streamlit run app.py
